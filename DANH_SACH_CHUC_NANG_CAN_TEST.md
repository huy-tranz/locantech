# Danh Sách Chức Năng Chính Cần Test

1. Sản phẩm
- Danh sách sản phẩm
- Chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Lọc sản phẩm
- Sắp xếp sản phẩm
- Hiển thị giá, khuyến mãi, tình trạng hàng
- Sản phẩm liên quan

2. Danh mục và điều hướng
- Trang chủ
- Trang danh mục sản phẩm
- Menu header
- Footer
- Breadcrumb nếu có
- Điều hướng giữa các trang

3. Giỏ hàng và đặt hàng
- Thêm sản phẩm vào giỏ hàng
- Cập nhật số lượng sản phẩm
- Xóa sản phẩm khỏi giỏ hàng
- Tính tổng tiền
- Nhập thông tin đặt hàng
- Xác nhận đặt hàng
- Trạng thái đặt hàng thành công hoặc thất bại

4. Liên hệ và tư vấn
- Trang liên hệ
- Form liên hệ
- Gửi yêu cầu tư vấn
- Thông tin hotline, email, địa chỉ
- Bản đồ nếu có

5. Tài khoản người dùng nếu có
- Đăng ký tài khoản
- Đăng nhập
- Đăng xuất
- Quên mật khẩu
- Thông tin cá nhân
- Lịch sử đơn hàng

6. Trải nghiệm người dùng
- Giao diện desktop
- Giao diện tablet
- Giao diện mobile
- Tốc độ tải trang
- Trạng thái loading
- Trang thông báo lỗi
- Trang 404
- Tính nhất quán của màu sắc, font chữ, khoảng cách

7. Admin tổng quan
- Đăng nhập admin
- Đăng xuất admin
- Dashboard admin
- Phân quyền truy cập admin
- Bảo vệ các trang admin khi chưa đăng nhập

8. Admin quản lý sản phẩm
- Thêm sản phẩm
- Sửa sản phẩm
- Xóa sản phẩm
- Upload ảnh sản phẩm
- Quản lý giá sản phẩm
- Quản lý tồn kho
- Quản lý trạng thái hiển thị sản phẩm

9. Admin quản lý danh mục
- Thêm danh mục
- Sửa danh mục
- Xóa danh mục
- Sắp xếp danh mục
- Ẩn hoặc hiện danh mục

10. Admin quản lý đơn hàng
- Danh sách đơn hàng
- Chi tiết đơn hàng
- Cập nhật trạng thái đơn hàng
- Tìm kiếm đơn hàng
- Lọc đơn hàng
- Xóa hoặc hủy đơn hàng nếu có

11. Admin quản lý liên hệ
- Danh sách liên hệ
- Chi tiết liên hệ
- Cập nhật trạng thái xử lý liên hệ
- Xóa liên hệ

12. Nội dung và chính sách
- Trang giới thiệu
- Chính sách bảo hành
- Chính sách đổi trả
- Chính sách vận chuyển
- Chính sách bảo mật
- Câu hỏi thường gặp nếu có

13. Bảo mật cơ bản
- Bảo vệ route admin
- Kiểm tra quyền người dùng
- Kiểm tra form nhập liệu
- Kiểm tra lỗi khi truy cập sai quyền
- Kiểm tra token hoặc thông tin nhạy cảm

14. Dữ liệu và đồng bộ
- Dữ liệu sản phẩm trên website
- Dữ liệu sản phẩm trong admin
- Dữ liệu danh mục
- Dữ liệu đơn hàng
- Dữ liệu liên hệ
- Đồng bộ sau khi thêm, sửa, xóa trong admin

15. SEO và chia sẻ
- Tiêu đề trang
- Mô tả trang
- Đường dẫn URL
- Ảnh chia sẻ
- Sitemap nếu có
- Robots.txt nếu có
